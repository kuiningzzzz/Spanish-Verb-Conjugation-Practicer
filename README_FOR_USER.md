# Conjugamos 使用说明

## 1. 我们的产品是做什么的
Conjugamos 是一款面向西班牙语学习者的练习平台，核心目标是帮助你更高效地掌握动词变位。

你可以把它理解成一个“练习 + 课程 + 复盘 + 反馈”的完整学习工具：
- 想快速刷题：可以直接开始单词练习。
- 想系统学习：可以按教材、按课程推进。
- 想针对提升：可以用收藏、错题、滚动复习做强化。
- 想互动对比：可以使用好友和 PK 功能。

## 3. 用户端使用教程（学习者）

### 3.1 登录与进入
1. 打开 App 后，先完成登录/注册。
2. 登录成功后会进入首页。
3. 首页可直接进入：课程练习、单词练习、排行榜、学习统计、好友。

### 3.2 首页怎么看
首页会展示：
- 今日练习数、答对题数、已掌握动词数
- 连续打卡状态
- 快速入口（课程、单词、排行、统计、好友）

建议每天从首页进入，先做一轮短练习，保持连续打卡。

### 3.3 单词练习（自由练习）
1. 点击“单词练习”进入。
2. 选择练习模式后开始答题。
3. 练习中可以：
- 收藏单词
- （部分题型）收藏题目
- 查看提示/翻译
- 对题目进行错误反馈
4. 完成后查看本轮结果，系统会记录正确率和练习数据。

### 3.4 课程练习（教材学习）
1. 进入“课程练习”。
2. 先点击“添加教材”，把需要的教材加入自己的学习列表。
3. 展开教材后可以看到课程列表，每课可：
- 开始练习
- 滚动复习（综合复习前面课程）
- 展开查看本课单词
4. 单词右侧可查看完整变位。

### 3.5 单词本与错题本
在“单词本”里可进行针对性练习：
- 收藏专练：练你标记过的词
- 错题专练：练你曾经做错的词
- 收藏题目专练：练你收藏过的题

这三个入口非常适合考前复习。

### 3.6 变位查询
在“变位查询”页可直接搜索动词，查看完整变位与释义。
适合做“练前预习”和“练后查漏补缺”。

### 3.7 个人中心
在“我的”页面可以：
- 编辑个人信息
- 进入设置（练习偏好、界面偏好、隐私偏好）
- 查看学习成就
- 检查更新与更新日志
- 提交用户反馈

### 3.8 好友与互动
在“好友”模块可以：
- 添加好友
- 处理好友申请
- 设置备注与星标
- 查看好友学习卡片
- 进行数据 PK

如果你希望和同学一起学习，建议开启这块功能。

## 4. 管理端使用教程（Admin 前端）

> 管理端用于内容维护与教学管理，不是普通学习入口。

### 4.1 登录
1. 打开管理端登录页。
2. 使用后台账号登录（管理员账号）。
3. 登录后进入仪表盘。

### 4.2 仪表盘
仪表盘主要做两件事：
- 欢迎区：快速确认当前账号身份
- 管理总览：查看用户数、词库量、题库量、教材量并跳转对应页面

页面下半部分有“管理历史”，用于查看近期后台操作记录。

### 4.3 用户管理
可进行：
- 搜索、筛选、分页查看
- 编辑用户资料
- 角色与类型调整（受权限限制）

### 4.4 词库管理
可进行：
- 新建动词条目
- 编辑词义与字段
- 编辑变位
- 删除（受权限限制）

支持批量生成与队列处理，也支持按标签查看词条属性（如 Irreg. / Prnl. / tr. / intr. 等）。

### 4.5 题库管理
可进行：
- 按“传统变位 / 带代词变位”切换
- 搜索、排序、分页
- 编辑题目
- 删除（受权限限制）

### 4.6 课程教材管理（重点）
教材管理分两层：
- 教材（一本书）
- 课程（一本教材下若干课）

典型流程：
1. 在教材列表点击“添加教材”，只填教材名。
2. 进入“设置课程”，新增课程（自动生成中文课名：第一课、第二课...）。
3. 在每一课中设置：
- 单词（只能从词库中搜索后加入）
- 时态（可按栏位勾选、全选、清除）
4. 教材先以草稿保存；确认完整后点击“发布教材”。

当前页面还提供：
- 发布/取消发布状态切换
- 上传者显示
- 最近修改时间
- 删除教材（有二次确认）

权限规则：
- `admin` 仅可修改自己上传的教材。
- 对他人上传教材，按钮会变为“查看教材”，只能看不能改。

### 4.7 反馈处理
- `dev`：可处理用户反馈和题目反馈。
- `superadmin`：页面为“题目反馈”，不显示“用户反馈”选项。

### 4.8 班级管理与作业管理
当账号类型为“教师用户”时，可进入：
- 班级管理
- 作业管理

用于教学组织与作业分发。

## 5. 角色权限（通俗版）

### 5.1 admin
- 日常内容管理角色。
- 可维护用户、词库、题库、教材（教材仅限本人上传）。
- 看不到 superadmin/dev 用户列表。

### 5.2 superadmin
- 高级管理角色。
- 可用大部分高级后台能力。
- 反馈处理默认按题目反馈工作。

### 5.3 dev
- 开发维护角色。
- 拥有完整后台管理能力（含技术维护页面）。

## 6. 常见问题

### Q1：为什么我在用户管理里看不到某些账号？
不同角色看到的用户范围不同。比如 admin 看不到 dev/superadmin，这是权限设计。

### Q2：我可以给后台不可见的用户加好友吗？
可以。好友系统走的是用户端逻辑，不依赖后台“用户管理列表可见性”。

### Q3：教材什么时候会给学习者可用？
教材在草稿状态下不会给学习者添加，发布后才会在用户端可选。

### Q4：如果我改了课程名，会不会丢失？
课程名是自动保存的，不需要额外点保存。

### Q5：设置单词时可以手动输入新词吗？
不可以。课程单词只能从词库中搜索并加入。

## 7. 给新用户的建议学习路径
如果你是第一次使用，建议按下面顺序：
1. 先做 1 轮“单词练习”，熟悉答题节奏。
2. 再进入“课程练习”，添加教材并按课学习。
3. 每天结束前做“错题专练”5-10 分钟。
4. 每周看一次“学习统计”，调整练习重点。
5. 遇到问题及时用“用户反馈”提交。

坚持短时高频，会比长时间低频更有效。
